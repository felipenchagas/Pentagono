Software Info - Technical

Main Features
Responsive HTML5 system via Twitter Bootstrap
Add categories and journals easily via attractive admin interface
Easily add menu boxes, new pages and themes for layout changes
Simple API to open journals from any application
SMTP settings
Utilises the latest technology and built with modern standards
SEO options such as alternative browser titles and search engine friend urls
Database backup
Add private journals and categories
Set journals and / or categories to be auto deleted at a certain date / time
Set journals to publish automatically in the future
Add staff members with different privileges
Pin journals to top of list screens if important
Send tweet to your Twitter account when adding journal
..and other great features..
